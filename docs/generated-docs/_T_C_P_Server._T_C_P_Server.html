TCPServer

TCPServer

The purpose of the TCPServer is to accept multiple client connections. Every client that connects will have its own thread for the server to broadcast information to each client.

class TCPServer {

Client[] mClientsConnectedToServer;
char[80][] mServerData;
uint[] mCurrentMessageToSend;
int clientCount;
~this();
void run();
void broadcastToAllClients();
